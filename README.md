# Komoreby-Game
Komorebi is a cross-platform mobile application developed in a visual novel format, specifically designed to teach Japanese social norms and cultural etiquette to students and language enthusiasts.

Through gamification and interactive decision-making, users immerse themselves in daily life scenarios in Japan, learning how to react in a culturally appropriate manner across various contexts.

Key Features
Interactive Visual Novel: A branching narrative where player choices directly impact the story progression and evaluate their understanding of Japanese social protocols.

Real-Time Cultural Assessment: Built-in logic that analyzes and scores the user's behavior and responses within specific socioeconomic and cultural contexts.

Cross-Platform Architecture: A robust build designed for smooth, native performance across multiple device platforms.

Tech Stack
The core architecture of the project was built using modern, industry-standard technologies for scalable software development:

Frontend & Logic: Flutter & Dart (For efficient cross-platform development and native performance).

Database & Backend: Cloud Firestore (A NoSQL real-time database used for dynamic user management, progress tracking, and game-state persistence).

AI Engine Integration: Seamless integration with modern AI capabilities to drive the real-time cultural evaluation engine.
